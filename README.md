# PoCN Projects

Short guide to the repo layout:

- `report.pdf`: compiled report for submission.

- `latex/`: the report (start from `latex/main.tex`).
- `code/task_15/`: Task 15 SOC code and reproduction steps.
- `code/task_44/`: Task 44 (Social Connectedness Index) code and docs.
- `data/task_15/`: Task 15 outputs used in the report.
- `data/task_44/`: Task 44 inputs and outputs.

If you only want the final deliverable and figures, open the report in `latex/` and follow the figures/sections from there.
